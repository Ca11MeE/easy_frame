# easy_frame
> python 基于Flask的后台框架

### 功能简述:
#### 实例参照配置自动注入
#### 数据库连接池
#### 配置形式维护参数
#### 远程xml维护
#### binlog检查增量(调度形式)
#### sql语句分页,变量赋值,批量执行,细粒度事务,调试模式执行详情打印
#### 一键启动(类似springboot)
#### 蓝图形式绑定路径
#### 装饰器形式数据返回(ResponseBody)

- 运行环境:
  python3.6
- 所需模块:
  flask
  pymysql
  schedule
  pyOpenSSL(可选)
